# Скрипт заполнения БД тестовыми данными

## Описание

`seed_test_data.py` - скрипт для заполнения локальной базы данных тестовыми профилями пользователей для полноценного тестирования бота TeammateBot.

## Что создает скрипт

### Пользователи и профили
- **Уникальные telegram_id** (9-значные числа)
- **Реалистичные русские имена** (комбинации имен и фамилий с правильными окончаниями)
- **Игровые никнеймы** в стиле геймеров (ProGamer, ShadowKiller228, MegaSniper_uwu и т.д.)
- **Username** в стиле Telegram или их отсутствие
- **Возраст** с реалистичным распределением (больше людей 18-27 лет)
- **Разнообразные характеристики**:
  - Рейтинги (нормальное распределение - больше в середине)
  - Позиции (1-3 позиции на профиль)
  - Регионы (70% СНГ, 30% остальной мир)
  - Цели (1-3 цели)
  - Роли (игрок/тренер/менеджер)
  - Дополнительная информация (или её отсутствие)

## Использование

### Базовый запуск
```bash
# Активируем виртуальную среду
source .venv/bin/activate

# Запускаем с параметрами по умолчанию (50 профилей для каждой игры)
python seed_test_data.py
```

### Запуск с параметрами
```bash
# Создать 100 профилей для Dota 2 и 100 для CS2
python seed_test_data.py --dota 100 --cs 100

# Создать только профили для Dota 2
python seed_test_data.py --dota 200 --cs 0

# Создать только профили для CS2
python seed_test_data.py --dota 0 --cs 150
```

### Параметры командной строки
- `--dota N` - количество профилей для Dota 2 (default: 50)
- `--cs N` - количество профилей для CS2 (default: 50)

## Требования

### База данных
Перед запуском убедитесь, что:
1. PostgreSQL и Redis запущены
2. В `.env` настроены правильные параметры подключения:
   ```
   DB_HOST=localhost
   DB_PORT=5432
   DB_NAME=teammates
   DB_USER=teammates_user
   DB_PASSWORD=your_password
   ```

### Запуск БД через Docker
```bash
docker-compose up -d
```
